Mentions légales


L'application OC Inventory est un projet Openclassrooms créé par David Bouzerar dans le cadre du parcours - développeur d'application Python. Elle a été développé, principalement, en Python et s'appuie, en grande majorité, sur le framework Django.

L'aspect visuel est assuré par HTML5, Javascript et CSS (framework Bootstrap) L'intégration des graphiques et des tableaux dynamiques se fait par l'intermédiaire des librairies Javascript ChartJS et Simple-Datatables.

Hébergeur

L'application OC Inventory est hébergée sur la plateforme DigitalOcean, dont le siège se situe à New York City, New York, USA

Ressources

L'application web utilise différentes ressources (principalement pour les icônes) dont voici les références :

  • IcoMoon
  • FontAwesome

Contenu : Les informations contenues dans les bases de données ont été générées par le biais de l'outil en ligne Mockaroo

Vie privée

Les données renseignées lors de la création d'un compte utilisateur ne requièrent pas d'informations privées et ne sont là que dans un but de démonstration des possibilités offertes par l'application. L'adresse mail est requise, uniquement, pour valider et activer le compte utilisateur lors de l'inscription. Si toutefois l'utilisateur souhaite supprimer son compte, merci de le signaler dans le formulaire de contact : en indiquant dans l'objet "Suppression compte utilisateur". L'action sera effective sous 72h.
